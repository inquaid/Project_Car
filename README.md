# TikTakToe

A simple Tic Tac Toe game implemented in Java using Swing.

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/your-username/TikTakToe.git
   ```
2. Navigate to the project directory:
   ```
   cd TikTakToe
   ```
3. Compile the Java file:
   ```
   javac TikTakToe.java
   ```
4. Run the compiled Java class:
   ```
   java TikTakToe
   ```

## Usage

1. After running the program, a window will appear with a 3x3 grid and the title "Tic Tac Toe".
2. The game starts with a random player (X or O) going first.
3. Click on any of the buttons in the grid to make a move.
4. The game will alternate turns between X and O players.
5. The game will display the winner or a draw message once the game is over.

## Features

- Simple and intuitive GUI
- Randomly decides which player goes first
- Highlights the winning combination
- Displays the result (X wins, O wins, or Draw)
